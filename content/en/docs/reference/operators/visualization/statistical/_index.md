## Operators

| Operator | Description |
|----------|-------------|
| [Box/Violin Plot](boxviolin-plot/) | Visualize data using either a Box Plot or a Violin Plot. Box plots are drawn as a box with a vertical line down the middle which is mean value, and has horizontal lines attached to each side (known as “whiskers”). Violin plots provide more detail by showing a smoothed density curve on each side, and also include a box plot inside for comparison. |
| [Continuous Error Bands](continuous-error-bands/) | Visualize error or uncertainty along a continuous line |
| [Histogram](histogram/) | Visualize data in a Histogram Chart |
| [Histogram2D](histogram2d/) | Displays a bivariate histogram as a density heatmap |
| [Scatter Matrix Chart](scatter-matrix-chart/) | Visualize datasets in a Scatter Matrix |
| [Strip Chart](strip-chart/) | Visualize distribution of data points as a strip plot |
| [Tree Plot](tree-plot/) | Visualize hierarchical data as a top-down, interactive, auto-sizing tree |
