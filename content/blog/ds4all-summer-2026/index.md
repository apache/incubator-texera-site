<p style="font-family: Georgia,'Times New Roman',serif; font-size: 21px; line-height: 1.5; margin: 0 0 20px;">
The fourth <a style="color: #c8451f; text-decoration: none; font-weight: bold;" href="https://ds4all.ics.uci.edu/summer2026/" target="_blank" rel="noopener">Data Science for All (DS4ALL)</a> summer program wrapped up at UC Irvine, bringing high-school students on campus for a week of hands-on data science and machine learning &mdash; taught almost entirely through the open-source Apache Texera (Incubating) platform.
</p>

<p style="font-family: Georgia,'Times New Roman',serif; font-size: 21px; line-height: 1.5; margin: 0 0 20px;">
The premise of DS4ALL is simple: students should not need years of programming experience before they can do real data science. With a &ldquo;no code and low code&rdquo; approach built on visual workflows, participants spent the week working with actual datasets rather than waiting to earn the privilege.
</p>

<h2 style="font-family: Georgia,'Times New Roman',serif; font-weight: 900; font-size: 32px; letter-spacing: -.01em; margin: 48px 0 6px; line-height: 1.1; color: #14110f;">
<span style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 14px; font-weight: 800; color: #c8451f; letter-spacing: .1em; border: 2px solid #c8451f; border-radius: 999px; padding: 3px 11px; margin-right: 10px;">01</span>
What the Program Covers
</h2>

<div style="height: 3px; width: 60px; background: #14110f; margin: 0 0 22px;"></div>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
Over the week, students worked through the core concepts that underpin practical data science: data modeling, data cleaning, data wrangling, and visualization. From there the program moves into basic Python programming for data processing, and then into AI and machine learning techniques &mdash; building and training models, running inference, and working with classification and clustering.
</p>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
Each day pairs a morning lecture with an afternoon lab session, so concepts introduced in the morning get applied the same day. The week builds toward a capstone project in which students analyze real data end to end and present their results at a final showcase, with family members invited to attend.
</p>

<div style="background: #eef2f0; border-left: 4px solid #1f6b5c; padding: 18px 22px; margin: 24px 0; font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 16px; border-radius: 0 8px 8px 0;">
Students who complete the program without any absence receive a certificate, and those with strong performance are eligible for recommendation letters from the organizers.
</div>

<h2 style="font-family: Georgia,'Times New Roman',serif; font-weight: 900; font-size: 32px; letter-spacing: -.01em; margin: 48px 0 6px; line-height: 1.1; color: #14110f;">
<span style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 14px; font-weight: 800; color: #c8451f; letter-spacing: .1em; border: 2px solid #c8451f; border-radius: 999px; padding: 3px 11px; margin-right: 10px;">02</span>Why Texera
</h2>

<div style="height: 3px; width: 60px; background: #14110f; margin: 0 0 22px;"></div>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
The barrier for a ninth grader entering data science is rarely the ideas &mdash; it is the toolchain. Environment setup, dependency management, and boilerplate code can consume the entire week before a student ever sees a result.
</p>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
Texera's visual workflow model removes most of that overhead. Students assemble pipelines by connecting operators on a canvas, see intermediate results at every step, and drop into Python User-Defined Functions when they want finer control. The same platform that supports research workloads scales down to a classroom without becoming a toy, which means students learn concepts that transfer directly to professional practice.
</p>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
This also makes DS4ALL a genuinely useful source of feedback for the project. A cohort of first-time users working through a full data science lifecycle in five days surfaces rough edges that experienced contributors have long since learned to navigate around.
</p>

<h2 style="font-family: Georgia,'Times New Roman',serif; font-weight: 900; font-size: 32px; letter-spacing: -.01em; margin: 48px 0 6px; line-height: 1.1; color: #14110f;">
<span style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 14px; font-weight: 800; color: #c8451f; letter-spacing: .1em; border: 2px solid #c8451f; border-radius: 999px; padding: 3px 11px; margin-right: 10px;">03</span>Who Runs It
</h2>

<div style="height: 3px; width: 60px; background: #14110f; margin: 0 0 22px;"></div>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
DS4ALL is organized by faculty from two campuses &mdash; <b>Prof. Chen Li</b> (Computer Science, UC Irvine) and <b>Prof. Wei Wang</b> (Computer Science, UCLA) &mdash; with instructors and staff drawn from professors and Ph.D. students at UCI and UCLA who work in data management, data science, AI, and machine learning.
</p>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
The program is open to students entering 9th, 10th, or 11th grade, with Algebra II or Integrated Math II as the only prerequisite. Sessions run 9 AM to 4 PM daily on the UCI campus, with lunch provided.
</p>

<h2 style="font-family: Georgia,'Times New Roman',serif; font-weight: 900; font-size: 32px; letter-spacing: -.01em; margin: 48px 0 6px; line-height: 1.1; color: #14110f;">
<span style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 14px; font-weight: 800; color: #c8451f; letter-spacing: .1em; border: 2px solid #c8451f; border-radius: 999px; padding: 3px 11px; margin-right: 10px;">04</span>A Growing Outreach Effort
</h2>

<div style="height: 3px; width: 60px; background: #14110f; margin: 0 0 22px;"></div>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
DS4ALL is now in its fourth year, and it sits alongside a broader set of education and outreach efforts that use Texera to lower the entry cost of data science &mdash; including workshops for middle school students and summer camps aimed at biology students with limited coding backgrounds.
</p>

<p style="font-family: 'Helvetica Neue',Arial,sans-serif; font-size: 17px; margin: 0 0 18px;">
Making advanced data science and AI accessible to people who are not primarily programmers is one of the reasons Apache Texera exists. Watching a room of high schoolers build and present working machine learning pipelines in five days is a reasonably direct test of whether that goal is being met.
</p>
