Join the Apache Texera community

Apache Texera is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Apache Texera? Find out more here:

  • User mailing list: Discussion and help from fellow users
  • GitHub: Questions, ideas, and Q&A

Develop and Contribute

If you want to get more involved by contributing to Apache Texera, join us here:

  • YouTube: Development discussions
  • Issues: Report bugs and track features

You can find out how to contribute to Apache Texera in our Contribution Guidelines.
