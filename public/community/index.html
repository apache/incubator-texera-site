Join the Texera community

Texera is an open source project that anyone in the community can use, improve, and enjoy. We'd love you to join us! Here's a few ways to find out what's happening and get involved.

Learn and Connect

Using or want to use Texera? Find out more here:

  • User mailing list: Discussion and help from fellow users
  • Announcements: ASF-wide release and news announcements
  • GitHub Discussions: Questions, ideas, and Q&A

Develop and Contribute

If you want to get more involved by contributing to Texera, join us here:

  • GitHub: Development happens here
  • Dev mailing list: Development discussions
  • Issues: Report bugs and track features

You can find out how to contribute to Texera in our Contribution Guidelines.
