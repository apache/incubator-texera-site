Apache Texera
(Incubating)

Apache Texera (Incubating) is an open-source system for human-AI collaborative data science using visual workflows. It enables analysts to construct, execute, and refine data analysis tasks through an intuitive GUI, assisted by AI agents that understand natural-language instructions. Texera is well suited for a wide range of applications, including “AI for Science,” by making advanced AI and data science capabilities accessible to a broader community. It can run on a laptop for local use or be deployed in the cloud to support scalable processing of large datasets.

Capabilities

Operator 01

AI-Powered Data Science

Active
chatbots natural-language no-code

Conduct data science through AI chatbots using natural language — no coding required. AI agents help you build, refine, and execute analytical workflows conversationally.

Operator 02

Visual Workflow Interface

Pending
drag-drop intuitive-gui low-code

Construct end-to-end data analysis pipelines through an intuitive GUI. Drag-and-drop operators, visualize intermediate results, and debug workflows interactively at runtime.

Operator 03

Scalable Processing Engine

Pending
parallel-engine big-data compute-storage-separation

Process large volumes of data with a parallel backend engine. Supports separation of compute and storage for flexible, high-performance execution at scale.

Operator 04

Real-Time Collaboration

Pending
collaborative-editing shared-execution multi-tenant

Work together like Google Docs, but for data science. Co-edit workflows, share executions in real time, and manage multiple tenants with workload isolation.

Operator 05

Multi-Language UDFs

Pending
python r-lang java scala

Extend workflows with user-defined functions in Python, R, Java, and Scala. Go from no-code to full-code seamlessly within the same workflow environment.

Operator 06

Cloud-Native Deployment

Pending
kubernetes cloud-services extensible

Deploy on a laptop for local use or scale to the cloud with Kubernetes. Extensible architecture integrates external web services and grows with your data ecosystem.
